# 🌱 Dry Bean Classification using Machine Learning

## 📌 Project Overview

The **Dry Bean Classification System** is a Machine Learning web application that predicts the variety of a dry bean based on its physical characteristics. The application is built using **Python**, **Scikit-learn**, and **Streamlit**, providing an interactive interface where users can enter bean measurements and instantly receive a prediction along with the model's confidence.

This project demonstrates the complete Machine Learning workflow, including data preprocessing, feature scaling, model training, evaluation, and deployment.

---

## 📊 Dataset

The dataset contains measurements of different dry bean varieties along with their corresponding class labels.

### Features

* Area
* Perimeter
* Major Axis Length
* Minor Axis Length
* Aspect Ratio
* Eccentricity
* Convex Area
* Equivalent Diameter
* Extent
* Solidity
* Roundness
* Compactness
* Shape Factor 1
* Shape Factor 2
* Shape Factor 3
* Shape Factor 4

### Target Classes

* BARBUNYA
* BOMBAY
* CALI
* DERMASON
* HOROZ
* SEKER
* SIRA

* Interactive Streamlit web interface
